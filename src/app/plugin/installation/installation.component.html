Installation of the plugin

  1. Download the latest version from the following link: https://github.com/Salvatore-Als/cs2-rank/releases
  2. After downloading, drag and drop the 'addons' folder into the '/game/csgo/' directory at the root of your installation.
  3. Next, download the MYSQL MM plugin created by poggu, available at the same link.
  4. Perform the same installation steps for the MYSQL MM plugin.

Note: The plugin requires a MYSQL database for proper functionality.
