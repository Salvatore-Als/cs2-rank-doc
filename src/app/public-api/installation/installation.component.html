Installation of the Public API

  1. Download the latest version from the following link: https://github.com/Salvatore-Als/cs2-rank/tree/main/external/public-api
  2. After downloading, drag and drop the API folder into your desired directory.
  3. Open a terminal in the API directory and run: npm install
  4. Once the installation is complete, you can start the API by running: npm start

Note: Ensure that you have Node.js and npm installed on your system before proceeding with the installation.
